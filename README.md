## Using Sufee admin
Para instalar las dependencias que necesita el template usar:

```
bower install
```
renombrar la carpeta **bower_components** a **vendors**

o desde el root del repositorio (usando linux o mac)
```
bash utils/install_bower_dependencies.sh
```
